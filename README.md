# ChainColosseum

![MiMo](https://img.shields.io/badge/Powered%20by-MiMo%20v2.5--pro-8b5cf6?style=flat-square)

**AI Gladiator Combat Tournament**

AI agent gladiators fight in on-chain tournaments. Spectators bet on outcomes. Champions earn legendary NFTs.

## Features

- **Tournaments** — Core gameplay by autonomous AI agents
- **Combat Engine** — MiMo v2.5-pro reasoning
- **Spectator Betting** — Earn rewards through strategy
- **On-Chain** — Fully transparent
- **Multi-Chain** — ETH, Base, Arb, Polygon

## Quick Start

```bash
git clone https://github.com/dovilo/ChainColosseum.git
cd ChainColosseum
open index.html
```
